COMS21202 - Symbols, Patterns and Signals

Unit Information

This unit seeks to acquaint you with the fundamental aspects of processing digital data, presented in the context of concrete examples from applications in computer vision, graphics, speech, audio, machine learning and data mining. Particular emphasis is placed on the importance of representation and modelling.

Installation Instructions:

Jupiter Notebook - For all COMS21202 needs you are encouraged to install Anaconda (Python 2.7) or (Python 3.5) as it bundles all the course's requirements. Alternatively for manual installation, you will require Python 2.7.x with 'Jupyter' and 'iPython' both possibly in version 4.x.x as well as the following packages: 'Pillow', 'scikit-image', 'matplotlib', 'numpy', 'scipy', 'scikit', 'pygments' and 'scikit-learn'

In the lab Windows machines, before running Jupyter Notebook you need to run
$ conda install pillow
